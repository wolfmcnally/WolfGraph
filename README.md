A Swift-based general graph structure with value semantics.

## Requirements

Swift 4.2

## Installation

WolfGraph is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WolfGraph'
```
